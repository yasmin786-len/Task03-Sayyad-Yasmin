## Project 3: Recommendation System

A content-based recommendation system that suggests courses based on user interests using TF-IDF Vectorization and Cosine Similarity.

### Features

* User Interest Input
* TF-IDF Vectorization
* Cosine Similarity
* Ranked Recommendations

### Technologies Used

* Python
* Scikit-Learn
